Project Description

This project implements an embedded-system-based solar tracking system using an Arduino Mega 2560. The system uses four LDR sensors to detect the direction of the strongest light source and adjusts two servo motors to track the light automatically. A finite state machine was implemented with OFF, IDLE, ACTIVE, and ERROR states. LEDs indicate the current system state, while buttons provide START, RESET, and OFF control functionality.

The project also demonstrates:

Interrupt implementation
Sensor integration
Servo actuator control
State-machine design
UART/Serial logging
Embedded systems programming concepts
